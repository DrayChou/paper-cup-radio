# Paper Cup Radio Testing Guide

## 目标

验证手机输入到桌面总台这条主链路是否稳定。

## 测试前准备

- 电脑和手机处于同一局域网
- 电脑端已打开 Paper Cup Radio
- 手机浏览器可访问首页 `/`

## 基础流程

1. 手机打开首页
2. 给纸杯起一个名字
3. 输入一段文本并发送
4. 电脑端检查：
   - 总台是否收到新播报
   - 是否自动复制到本机剪贴板
   - 历史记录里是否有该条内容

## 建议测试项

### 1. 单设备基础发送
- 连续发送 3 到 5 条短文本
- 检查顺序是否正确

### 2. 长文本发送
- 发送多行文本
- 检查换行是否保留

### 3. 随机命名体验
- 多次点击“换个名字”
- 检查命名是否自然、有变化

### 4. 多设备并发
- 两台手机分别命名并发送
- 检查来源设备名是否正确

### 5. 主题切换
- 手机端切换深浅色
- 总台切换深浅色
- 刷新后检查本地保存是否生效

### 6. 竖版窗口
- 将总台窗口调成偏竖版
- 检查底部悬浮分页按钮是否清晰可点

### 7. 剪贴板
- 检查自动复制是否成功
- 手动点击再次复制是否成功
